The instructions to run are as follows:
1. Make sure the correct modules are installed by executing this in terminal: pip install gspread google-auth

2. Attached to the project submission is a file labeled key.json. The JSON key must also be placed in the same folder as the source code. If for some reason it does not work, a new JSON code can be created by logging into the google account created for this project (included in final writeup).

Then, access the Google Cloud console, and go to Menu -> APIs & Services -> Credentials -> Create Credentials.
A new JSON key will be autodownloaded. Rename this as key.json and keep in the source code folder.

If all else fails, download the .csv directly from here: 
https://docs.google.com/spreadsheets/d/1_3pTBJ4FE_9h_2rRM-5GXTPmuSc4UESqrP-Z3Fx3ZxU/edit?usp=sharing

3. Run the code, and it should output the correct pairings.
